# Sistemas Operativos 
## ¿Que es un sismetema Operativo?
Un sistema operativo es un conjunto de programas que permite manejar la memoria, disco, medios de almacenamiento de información y los diferentes periféricos o recursos de nuestra computadora, como son el teclado, el mouse, la impresora, la placa de red, entre otros.

Este mismo aprovecha del _Hardware_ y del _Software_ para que las herramientas a utilizar funcionen.

## Componentes de un Sistema Operativo
- Aplicaciones de usuario 
- Interpretacion de comandos (CLI)/ interfaz grafica de usuario (GUI)
- Llamadas al sistema 
- Hal _Hardware_ _Abstraction_ _Layer_
### Aplicaciones del Sistema
- Juegos 
- Multimedia 
- Ofimatica 
- Desarrollo de _Software_
### Llamadas del Sistema 
- Sistemas de ficheros
- Comunicaciones 
- Grafico/Sonido 
- API POSIX
### Kernel 
- Controladores de _Hardware_
- Gestion de procesos
- Getion memoria
